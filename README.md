# Twitter Sentiment Analysis

## Project Overview

This project performs sentiment analysis on Twitter data using Natural Language Processing (NLP) and Machine Learning techniques. The goal is to classify tweets into positive, negative, or neutral sentiments.

## Objectives

* Analyze sentiment from Twitter text data
* Perform text preprocessing and data cleaning
* Convert text into numerical features using TF-IDF
* Train a Machine Learning model for sentiment classification
* Evaluate model performance
* Visualize sentiment distribution and results

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Dataset Features

* clean_text
* category (Positive, Negative, Neutral)

## Methodology

1. Data Loading
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction using TF-IDF
5. Train-Test Split
6. Model Training using Naive Bayes
7. Prediction and Evaluation
8. Visualization

## Visualizations

* Sentiment Distribution Graph
* Confusion Matrix

## Model Performance

* Machine Learning Algorithm: Multinomial Naive Bayes
* Accuracy Achieved: Approximately 73%

## Key Findings

* Sentiment analysis successfully classified tweets into sentiment categories.
* TF-IDF effectively transformed text into machine-readable features.
* The model achieved satisfactory classification performance.
* The project demonstrates practical applications of NLP in social media analytics.

## Conclusion

This project demonstrates how Natural Language Processing and Machine Learning can be used to analyze public opinion from social media data. The trained model achieved approximately 73% accuracy and effectively classified tweet sentiments.
